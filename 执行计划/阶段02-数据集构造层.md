**Goal:** 在研究数据底座之上，落地监督标签、样本索引、shift state/strata 数据产品和不可变 `dataset_manifest`，使研究数据可以被独立审计、预览和复用。

**Architecture:** 继续沿用并行 `research_platform` 包，在 `dataset/` 子模块下实现标签聚合、样本索引、shift state、strata 切分和 manifest 构造。结果通过 `/api/research-platform/datasets*` 暴露，不直接训练模型。

**Tech Stack:** FastAPI、SQLite、现有 `DataStorage`、pytest、pytest-asyncio、纯 Python 时间窗口聚合。

**Frozen Protocol Locks:**
- `label_definition_version_v1 = next_bar_15m_ohlc_reparam_from_session_seconds_v1`，标签对象必须保存：`target_id`、`session_id`、`inst_id`、`decision_ts`、`anchor_second_bucket`、`anchor_close_price`、`label_start_ts`、`label_end_ts`、`open_price`、`high_price`、`low_price`、`close_price`、`r_open`、`r_close`、`u`、`d`、`label_complete`、`invalid_reason`、`label_definition_version`。
- `sample_filter_rule_v1 = single_session_strict_7200x900_v1`，样本对象必须保存：`sample_id`、`session_id`、`inst_id`、`decision_ts`、`input_start_ts`、`input_end_ts`、`label_start_ts`、`label_end_ts`、`input_second_count`、`label_second_count`、`input_complete_7200`、`label_complete_900`、`sample_valid`、`ready_for_inference`、`ready_for_training`、`invalid_reason`、`prev_sample_overlap_seconds`、`stride_seconds`。
- `feature_recipe_version_v1 = second_state_causal_tensor_v1`，其语义是“输入来自单会话内连续 7200 秒 `second_state` 序列，不跨会话拼接”。
- `deployment_target_version_v1 = single_inst_all_deployment_eligible_15m_v1`，其语义是“该合约研究时段内全部 deployment-eligible 的 15m 边界”。
- `target_census_policy_version_v1 = deployment_eligible_boundary_census_v1`，与阶段 01 的 census 固定版本一致。
- `target_window_policy_version_v1 = expanding_pre_origin_census_v1`，语义固定为“outer origin 之前全部历史 census 扩展窗；任何 origin 只能使用 `decision_ts < origin` 的 census；strata 切点按 origin 单独拟合并冻结到 `strata_fit_ref`”。
- `strata_fit_ref` 不能是单个 origin 的切点快照；它必须指向按-origin 聚合的 artifact bundle，例如 `artifact://dataset/{dataset_id}/strata-fit-by-origin.json`，其中每个 `origin_ts` 分别保存 `rv_2h_bin_3` 与 `liquidity_bin_3` 的真实 cutpoints。
- `shift_assumption_version_v1 = A_shift_suff_v1`，用于标记“weighted deployment proxy 仅在声明好的 `S_t` 空间中、且以工作假设成立为前提”。
- `shift_diagnostic_version_v1 = support_mmd_propensity_v1`，最低诊断集固定为：strata coverage、MMD、blocked session-propensity classifier。
- `weight_fit_ref` 与 `domain_classifier_fit_ref` 也必须是按-origin bundle；前者至少保存每个 origin 的 `p_s / q_s` 频率表、有效样本量和最终权重摘要，后者至少保存每个 origin 的 scaler 参数、logistic 系数、intercept 和 OOF 概率摘要。
- `weighting_version_v1 = strata_ratio_weighting`，`weight_estimator_version_v1 = exact_strata_ratio_v1`，`weight_definition_v1 = raw_ratio_no_clip_no_self_normalization`。
- `weighting_version_v2 = classifier_density_ratio_weighting`，`domain_classifier_version_v1 = l2_logistic_shift_state_v1`，`weight_estimator_version_v2 = oof_logistic_odds_ratio_v1`，`weight_definition_v2 = raw_odds_ratio_no_clip_no_self_normalization`。
- `classifier_density_ratio_weighting` 的按-origin artifact 还必须附带：`class_prior_pi`、`sampling_scheme`、`calibration_method`；第一版固定 `class_prior_pi = 0.5`、`sampling_scheme = balanced_1_to_1`、`calibration_method = none`。
- `split_definition_version_v1 = blocked_temporal_hv_v1`，`embargo_sec` 最小值固定为 `8100`。
- `sampling_stride_sec` 只允许作用于训练段子采样；验证和测试段必须尽量回到全部 15m 边界评估，不能把 stride 透传到 val/test。
- `policy_parameter_ref` 与 `utility_parameter_ref` 在 `dataset_manifest` 中表示 outer-test 前冻结的参数协议对象；若运行时按 origin 重估，实际取值必须在阶段 03 的 `training_run` 中以 run-local、origin-keyed bundle ref 落地。
- `evaluation_protocol_version_v1 = rolling_origin_v1`，`score_definition_version_v1 = joint_scores_v1`，`prerank_definition_version_v1 = multicalibration_v1`，`regime_definition_version_v1 = boundary_regimes_v1`，`bootstrap_definition_version_v1 = stationary_block_bootstrap_min9_v1`，`multiple_comparison_version_v1 = locked_candidate_set_v1`。
- `boundary_regimes_v1` 不能停留在版本名；dataset preview 必须返回 regime schema，且 schema 至少包含 `hour_of_day`、`day_of_week`、`realized_vol_bin`、`spread_bin`、`liquidity_bin`、`funding_regime` 六个分层字段及其输出字段定义。
- `n_eff` 协议在 dataset 层就要写死：截断规则固定为 `initial_positive_sequence_v1`；preview 中必须显式声明三类序列定义 `primary_validation_score_sequence`、`label_r_close_sequence`、`model_comparison_delta_sequence`，其中 dataset 阶段只 materialize `label_r_close_sequence` 的 train/val/test 估计，另外两类留待阶段 03 的 training artifacts 生成。
- `dataset/manifest.py` 必须调用共享 `protocol_registry` 做白名单校验；任何不在冻结文档登记表中的版本字符串都不得写入 `dataset_manifest`。

@dataclass(frozen=True)
class BoundaryTarget15m:
    target_id: str
    session_id: str
    inst_id: str
    decision_ts: int
    anchor_second_bucket: int
    anchor_close_price: float
    label_start_ts: int
    label_end_ts: int
    open_price: float
    high_price: float
    low_price: float
    close_price: float
    r_open: float
    r_close: float
    u: float
    d: float
    label_complete: bool
    invalid_reason: str
    label_definition_version: str


def build_boundary_target_15m(*, session_id: str, inst_id: str, decision_ts: int, input_rows: list, label_rows: list, label_definition_version: str) -> BoundaryTarget15m:
    anchor_row = input_rows[-1]
    open_price = label_rows[0].open_price
    high_price = max(row.high_price for row in label_rows)
    low_price = min(row.low_price for row in label_rows)
    close_price = label_rows[-1].close_price
    anchor_close_price = anchor_row.close_price
    r_open = math.log(open_price / anchor_close_price)
    r_close = math.log(close_price / anchor_close_price)
    u = math.log(high_price / max(open_price, close_price))
    d = math.log(min(open_price, close_price) / low_price)
    return BoundaryTarget15m(
        target_id=f'{session_id}:{decision_ts}',
        session_id=session_id,
        inst_id=inst_id,
        decision_ts=decision_ts,
        anchor_second_bucket=decision_ts - 1,
        anchor_close_price=anchor_close_price,
        label_start_ts=decision_ts,
        label_end_ts=decision_ts + 900,
        open_price=open_price,
        high_price=high_price,
        low_price=low_price,
        close_price=close_price,
        r_open=r_open,
        r_close=r_close,
        u=u,
        d=d,
        label_complete=(len(label_rows) == 900),
        invalid_reason='' if len(label_rows) == 900 else 'label_seconds_missing',
        label_definition_version=label_definition_version,
    )
```

@dataclass(frozen=True)
class SampleIndex15m:
    sample_id: str
    session_id: str
    inst_id: str
    decision_ts: int
    input_start_ts: int
    input_end_ts: int
    label_start_ts: int
    label_end_ts: int
    input_second_count: int
    label_second_count: int
    input_complete_7200: bool
    label_complete_900: bool
    sample_valid: bool
    ready_for_inference: bool
    ready_for_training: bool
    invalid_reason: str
    prev_sample_overlap_seconds: int
    stride_seconds: int


def build_sample_index_15m(*, session_id: str, inst_id: str, decision_ts: int, input_rows: list, label_rows: list, stride_seconds: int) -> SampleIndex15m:
    input_second_count = len(input_rows)
    label_second_count = len(label_rows)
    input_complete_7200 = input_second_count == 7200
    label_complete_900 = label_second_count == 900
    sample_valid = input_complete_7200 and label_complete_900
    return SampleIndex15m(
        sample_id=f'{session_id}:{decision_ts}',
        session_id=session_id,
        inst_id=inst_id,
        decision_ts=decision_ts,
        input_start_ts=decision_ts - 7200,
        input_end_ts=decision_ts,
        label_start_ts=decision_ts,
        label_end_ts=decision_ts + 900,
        input_second_count=input_second_count,
        label_second_count=label_second_count,
        input_complete_7200=input_complete_7200,
        label_complete_900=label_complete_900,
        sample_valid=sample_valid,
        ready_for_inference=input_complete_7200,
        ready_for_training=sample_valid,
        invalid_reason='' if sample_valid else 'strict_window_incomplete',
        prev_sample_overlap_seconds=max(0, 7200 - stride_seconds),
        stride_seconds=stride_seconds,
    )
```

    errors = []

        value = payload[field_name]

def validate_weighting_preconditions(*, payload: dict[str, object], shift_diagnostic_result: dict[str, object]) -> None:
    if payload['weighting_version'] == 'strata_ratio_weighting':
        if payload['domain_classifier_version'] != '':
            raise ValueError('strata_ratio_weighting requires empty domain_classifier_version')
        if payload['weight_estimator_version'] != 'exact_strata_ratio_v1':
            raise ValueError('strata_ratio_weighting requires exact_strata_ratio_v1')
        return
    if payload['weighting_version'] != 'classifier_density_ratio_weighting':
        return
    if payload['domain_classifier_version'] != 'l2_logistic_shift_state_v1':
        raise ValueError('classifier_density_ratio_weighting requires l2_logistic_shift_state_v1')
    if payload['weight_estimator_version'] != 'oof_logistic_odds_ratio_v1':
        raise ValueError('classifier_density_ratio_weighting requires oof_logistic_odds_ratio_v1')
    overall_status = shift_diagnostic_result['overall_status']
    support_status = shift_diagnostic_result['checks']['support_overlap']['status']
    if overall_status != 'acceptable' or support_status != 'ok':
        raise ValueError('classifier weighting requires acceptable support overlap')
