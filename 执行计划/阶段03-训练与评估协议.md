**Goal:** 在 `dataset_manifest` 之上实现 `training_run`、rolling-origin 切分、weighting、forecast layer 评分、decision layer 效用与策略评估，并产出可查询的训练运行对象。

**Architecture:** 训练层完全依赖已冻结的数据产品层。新建 `training/` 子模块，包含 `protocols.py`、`splits.py`、`weighting.py`、`baselines.py`、`decision_policy.py`、`evaluation.py`、`bootstrap.py`、`comparison.py`、`service.py`。通过 `training_run` 表和 `/api/research-platform/training-runs*` 对外提供运行状态与结果。

**Tech Stack:** FastAPI、SQLite、NumPy、PyTorch（模型接入）、scikit-learn（审计友好的 L2 logistic DRE）、pytest、pytest-asyncio。

**Frozen Protocol Locks:**
- `split_definition_version_v1 = blocked_temporal_hv_v1`，切分必须按时间顺序，任意相邻块之间 `embargo_sec >= 8100`。
- `blocked_temporal_hv_v1` 不能只产出外层 `train/validation/test` 时间边界；`split_artifact_ref` 必须物化每个 outer origin 的 `inner_validation_folds`，并对每个 inner fold 保存：`fold_id`、`train_start_ts`、`train_end_ts`、`validation_start_ts`、`validation_end_ts`、`embargo_sec`、`origin_ts`。
- `evaluation_protocol_version_v1 = rolling_origin_v1`，外层结果至少报告 across-origin 均值、中位数、离散度和最差 origin。
- `refit_policy_version_v1 = expanding_refit_recompute_all_statistics_v1`，每个 outer origin 重新拟合 scaler、strata cutpoints、weighting；测试块内模型冻结。
- `score_definition_version_v1 = joint_scores_v1`，联合主评分固定为 `joint NLL`、`energy score`、`variogram score (p = 0.5, w_ij = 1)`；边际 CRPS 只做诊断。
- `prerank_definition_version_v1 = multicalibration_v1`，必须包含 `level_prerank`、`asymmetry_prerank`、`body_shadow_prerank`。
- `multivariate_rank_histogram` 第一版固定使用 `band_depth_rank_v1`，不再接受 `average-rank` / `band-depth rank` 二选一的模糊实现；若后续切换 rank 构造，必须升级协议版本。
- `bootstrap_definition_version_v1 = stationary_block_bootstrap_min9_v1`，块长不得小于 9 个 15m 边界。
- `multiple_comparison_version_v1 = locked_candidate_set_v1`，outer test 前冻结候选集合。
- `candidate_set_ref` 必须指向 outer test 开始前冻结的候选模型集合；若候选数大于 2，`comparison_result` 至少要给出 retained model set 或等价的多模型保留集合。
- `execution_assumption_version_v1 = boundary_rebalance_hold_to_close_v1`。
- `decision_utility_version_v1 = bar_close_return_with_adverse_excursion_penalty_v1`，必须显式回挂 `utility_parameter_ref`。
- `policy_definition_version_v1 = ternary_expected_utility_policy_v1`，必须显式回挂 `policy_parameter_ref`。
- `weighting_version_v1 = strata_ratio_weighting` 与 `weighting_version_v2 = classifier_density_ratio_weighting` 的 fit refs 必须分别持久化到 `weight_fit_ref` 与 `domain_classifier_fit_ref`。
- `classifier_density_ratio_weighting` 若启用，必须在 `domain_classifier_fit_ref` / `weight_fit_ref` 中按 origin 保存：`class_prior_pi`、`sampling_scheme`、`calibration_method`、`fold_protocol`、`eta_oof_summary`；第一版固定 `sampling_scheme = balanced_1_to_1`、`class_prior_pi = 0.5`、`calibration_method = none`。
- 所有 `weighted diagnostics` 一律先做归一化：`w_i^norm = w_i / sum_j w_j`；该归一化必须统一作用于 `joint score means`、`marginal coverage / weighted PIT`、`multivariate rank histogram` bin 计数、`pre-rank` 统计量、`regime` 分层均值与通过率。`unweighted diagnostics` 固定使用 `1 / n` 的均匀权重并明确标注为 `Q` 口径。
- `regime_definition_version_v1 = boundary_regimes_v1` 不能只有名字；输出 schema 固定包含 `hour_of_day`、`day_of_week`、`realized_vol_bin`、`spread_bin`、`liquidity_bin`、`funding_regime` 六个分层字段，并至少汇报 `regime_key`、`sample_count`、`normalized_weight_sum`、`joint_nll_mean`、`energy_score_mean`、`variogram_score_mean`、`calibration_error`、`sharpness_mean`。
- `n_eff` 协议固定采用 `initial_positive_sequence_v1` 截断规则，并至少对三类序列保存定义与估计：`primary_validation_score_sequence`、`label_r_close_sequence`、`model_comparison_delta_sequence`；训练运行不能只汇报一个泛化的 `effective_sample_size` 单值。
- `policy_parameter_ref` 与 `utility_parameter_ref` 在 `training_run` 中不能只复制 `dataset_manifest` 的默认值；若 inner validation 在各 origin 重估了参数，这两个字段必须更新为 run-local 的 origin-keyed bundle ref。
- 正式 challenger 的 `model_family` 只允许三类：`joint_density_model_v1`、`joint_sample_forecaster_v1`、`joint_quantile_scenario_model_v1`；`independent_four_head_regression_v1` 与 `point_forecast_head_v1` 只能作为 baseline family，不能注册为正式 challenger。
- `training_run` 不再接受一套独立协议参数；它必须从已冻结的 `dataset_manifest` 复制评估协议与数据协议，只额外接收模型配置和运行控制参数。
- `training/protocols.py` 必须调用共享 `protocol_registry` 做白名单校验，任何不在冻结文档登记表里的版本字符串都不得入库，也不得以“原样透传”方式绕过验证。

```python
        'decision_utility_version': manifest['decision_utility_version'],
        'execution_assumption_version': manifest['execution_assumption_version'],

def validate_model_output_contract(model_family: str) -> None:
    if model_family in {'joint_density_model_v1', 'joint_sample_forecaster_v1', 'joint_quantile_scenario_model_v1', 'joint_distribution_baseline'}:
        return
    if model_family in {'independent_four_head_regression_v1', 'point_forecast_head_v1'}:
        raise ValueError('non-joint model families can only be registered as baselines')
    raise ValueError(f'unknown model family: {model_family}')
```

```python
def build_blocked_temporal_hv_v1(*, decision_ts: list[int], embargo_sec: int, outer_origin_count: int) -> list[dict[str, int]]:
    ordered = sorted(decision_ts)
    origin_points = choose_outer_origins(ordered, outer_origin_count=outer_origin_count)
    splits = []
    for origin_ts in origin_points:
        pre_origin_ts = [ts for ts in ordered if ts < origin_ts]
        splits.append(
            {
                'origin_ts': origin_ts,
                'train_start_ts': pre_origin_ts[0],
                'train_end_ts': origin_ts - 2 * embargo_sec,
                'validation_start_ts': origin_ts - 2 * embargo_sec + 900,
                'validation_end_ts': origin_ts - embargo_sec,
                'test_start_ts': origin_ts + 900,
                'test_end_ts': choose_test_end_ts(origin_ts, ordered),
                'census_window_end_ts': origin_ts - 900,
                'inner_validation_folds': build_inner_blocked_validation_schedule(
                    pre_origin_ts=pre_origin_ts,
                    origin_ts=origin_ts,
                    embargo_sec=embargo_sec,
                    inner_fold_count=3,
                ),
            }
        )
    return splits


def build_inner_blocked_validation_schedule(*, pre_origin_ts: list[int], origin_ts: int, embargo_sec: int, inner_fold_count: int) -> list[dict[str, int]]:
    validation_blocks = partition_temporal_blocks(pre_origin_ts, block_count=inner_fold_count)
    schedule = []
    for fold_idx, validation_block in enumerate(validation_blocks):
        validation_start_ts = validation_block[0]
        validation_end_ts = validation_block[-1]
        schedule.append(
            {
                'fold_id': f'{origin_ts}:inner:{fold_idx}',
                'origin_ts': origin_ts,
                'train_start_ts': pre_origin_ts[0],
                'train_end_ts': validation_start_ts - embargo_sec,
                'validation_start_ts': validation_start_ts,
                'validation_end_ts': validation_end_ts,
                'embargo_sec': embargo_sec,
            }
        )
    return schedule


def compute_strata_ratio_weights(*, dataset_strata: list[str], census_strata: list[str], estimator_version: str, definition_version: str) -> dict[str, object]:
    p_counter = Counter(census_strata)
    q_counter = Counter(dataset_strata)
    p_total = sum(p_counter.values())
    q_total = sum(q_counter.values())
    missing_support = [key for key, count in p_counter.items() if count > 0 and q_counter.get(key, 0) == 0]
    weights = {
        key: (p_counter[key] / p_total) / (q_counter[key] / q_total)
        for key in q_counter
    }
    return {
        'weights': weights,
        'support_overlap_result': 'ok' if not missing_support else 'gap_detected',
        'dataset_status': 'ready' if not missing_support else 'research_only',
        'weight_fit': {
            'weight_estimator_version': estimator_version,
            'weight_definition': definition_version,
            'p_counter': dict(p_counter),
            'q_counter': dict(q_counter),
            'missing_support': missing_support,
        },
    }
```

```python
def fit_logistic_density_ratio_oof(*, rows: list[dict[str, object]]) -> dict[str, object]:

        clf.fit(X, feature_matrix['domain_labels'])
        class_prior_pi = 0.5
        origin_fit_bundle[str(origin_ts)] = {

            'coef': clf.coef_.tolist(),

            'class_prior_pi': class_prior_pi,
            'sampling_scheme': 'balanced_1_to_1',
            'calibration_method': 'none',
            'eta_oof_summary': summarize_probabilities(eta_oof),
        }
        origin_weight_bundle[str(origin_ts)] = {
            'eta_oof': eta_oof,

            'weight_formula': '((1-pi)/pi) * eta / (1-eta)',
```

```python
def compute_bar_utility(*, action: int, r_close: float, r_open: float, u: float, d: float, spread_last_bps: float, slippage_const_bps: float, previous_action: int, lambda_ae: float) -> float:
    high_return = max(r_open, r_close) + u
    low_return = min(r_open, r_close) - d
    turnover = abs(action - previous_action)
    execution_cost = (spread_last_bps / 2 + slippage_const_bps) / 1e4
    adverse_penalty = max(0.0, -low_return) if action == 1 else max(0.0, high_return) if action == -1 else 0.0
    return action * r_close - execution_cost * turnover - lambda_ae * adverse_penalty


def choose_ternary_action(*, expected_utilities: dict[int, float], previous_action: int, tau_entry: float, tau_switch: float) -> int:
    best_action = max(expected_utilities, key=expected_utilities.get)
    directional_best = max(expected_utilities[1], expected_utilities[-1])
    if directional_best <= tau_entry:
        return 0
    if best_action != previous_action and expected_utilities[best_action] - expected_utilities[previous_action] < tau_switch:
        return previous_action
    return best_action
```
