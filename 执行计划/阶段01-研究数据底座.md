**Goal:** 建立冻结文档要求的研究数据底座，完成 `collection_session`、`second_state`、`target_census_15m` 的后端存储、运行时服务、API 和 WebSocket 推送。

**Architecture:** 新建并行 `research_platform` 后端包，不侵入现有 `trend_research` 核心。平台由 `StorageResearchPlatformMixin`、`ResearchCollectionService`、`ResearchCensusService` 和 `ResearchPlatformService` 组成，通过独立 `/api/research-platform` 和 `research_platform` WebSocket 频道暴露能力。

**Tech Stack:** FastAPI、SQLite、现有 `DataStorage` 组合式存储、现有 OKX WebSocket 管理器、pytest、pytest-asyncio。

**Frozen Protocol Locks:**
- `collection_session` 必须显式保存：`session_id`、`inst_id`、`started_at`、`ended_at`、`planned_duration_sec`、`status`、`stop_reason`、`collector_version`、`source_config_hash`、`trigger_mode`、`trigger_note`、`sampling_policy_id`、`valid_second_count`、`missing_second_count`、`book_stale_second_count`、`state_stale_second_count`、`coverage_ratio`、`quality_score`。
- `second_state` 是 canonical source of truth，必须保存：`session_id`、`inst_id`、`second_bucket`、`ts_exchange`、`ts_local`、`bid_price`、`ask_price`、`bid_size`、`ask_size`、`mid_price`、`microprice`、`open_price`、`high_price`、`low_price`、`close_price`、`mark_price`、`index_price`、`trade_count`、`signed_trade_notional`、`buy_notional`、`sell_notional`、`buy_count`、`sell_count`、`max_trade_notional`、`buy_burst_count`、`sell_burst_count`、`buy_burst_notional`、`sell_burst_notional`、`open_interest`、`oi_delta`、`funding_rate`、`funding_delta`、`premium`、`basis_bps`、`spread_bps`、`book_level_count`、`multi_level_book_imbalance`、`book_slope`、`has_trade_input`、`has_book_input`、`has_state_input`、`book_age_seconds`、`state_age_seconds`、`clock_skew_ms`、`is_valid_second`、`quality_grade`、`invalid_reason`、`integrity_policy_version`。
- `target_census_15m` 必须显式保存：`census_id`、`inst_id`、`decision_ts`、`deployment_eligible`、`census_policy_version`、`shift_state_definition_version`、`shift_state_blob_json`、`hour_of_day`、`day_of_week`、`realized_vol_proxy_2h`、`spread_snapshot_bps`、`liquidity_snapshot_bin`、`funding_regime`、`session_active_flag`、`source_health_flag`、`invalid_reason`。
- `shift_state_definition_version_v1 = compact_boundary_state_v1` 必须逐字段持久化：`slot_15m`、`weekend_flag`、`ret_15m_bps`、`ret_2h_bps`、`rv_15m_bps`、`rv_2h_bps`、`range_15m_bps`、`range_2h_bps`、`spread_last_bps`、`spread_median_60s_bps`、`depth_10bps_log`、`imbalance_10bps`、`trade_count_60s`、`trade_notional_60s_log`、`funding_countdown_min`、`near_funding_flag`、`book_stale_ratio_60s`、`state_stale_ratio_60s`、`source_health_flag`、`session_active_flag`。
- `strata_definition_version_v1 = coarse_shift_strata_v1` 不在阶段 01 直接落库；阶段 01 只允许保存可审计的原始 `compact_boundary_state_v1`，禁止在未升版本的前提下把任何 `rv_2h_bin_3`、`liquidity_bin_3` 或主 strata 编码提前写入 census。
- `census_policy_version_v1` 在实现中固定为 `deployment_eligible_boundary_census_v1`，语义是“独立于手动会话的 15m 边界登记器，仅使用 `decision_ts = t` 时刻可观测信息”。

        await self._census.start()

        await self._census.stop()

async def record_second_state(self, session_id: str, *, second_bucket: int, payload: dict[str, object]) -> None:
    session = self._storage.get_research_collection_session(session_id)
    row = build_second_state_row(
        session_id=session_id,
        inst_id=session['inst_id'],
        second_bucket=second_bucket,
        payload=payload,
        integrity_policy_version=session['integrity_policy_version'],
    )
    self._storage.save_research_second_state(**row)
    self._storage.refresh_research_collection_quality(session_id)
    self._emit({'event': 'session_updated', 'session_id': session_id})


async def run_once(self, *, inst_id: str, decision_ts: int) -> None:

        inst_id=inst_id,
        decision_ts=decision_ts,
        census_policy_version='deployment_eligible_boundary_census_v1',

    )
    self._storage.save_research_target_census(**census_row)
    self._emit({'event': 'census_updated', 'inst_id': inst_id, 'decision_ts': decision_ts})
